# Secure FinTech API

A DevSecOps portfolio project demonstrating the architecture, security, and deployment of a financial API.

## 📌 Current Status
The project currently features the core architecture: a Python (FastAPI) application connected to a PostgreSQL database, containerized and orchestrated locally using Docker Compose.

## ⚙️ Prerequisites
To run this project locally, you need the following installed:
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)
- Git
